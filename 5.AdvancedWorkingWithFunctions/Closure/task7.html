7. Следующий код создаёт массив из стрелков (shooters).
Каждая функция предназначена выводить их порядковые номера. Но что-то пошло не так… 


        function makeArmy() {
            let shooters = [];
          
            let i = 0;
            while (i $lt; 10) {
              let shooter = function() { // функция shooter
                alert( i ); // должна выводить порядковый номер
              };
              shooters.push(shooter);
              i++;
            }
          
            return shooters;
          }
          
          let army = makeArmy();
          
          army[0](); // у 0-го стрелка будет номер 10
          army[5](); // и у 5-го стрелка тоже будет номер 10
          // ... у всех стрелков будет номер 10, вместо 0, 1, 2, 3...

Почему у всех стрелков одинаковые номера? Почините код, чтобы он работал как задумано.

Ответ: Выводится 10 из-за того, что при вызове функции shooter, так как внутри нет собственной переменной i и она берется из лексического окружения, но на тот момент цикл уже завершил свою работу и i равна 10. Поэтому номер всегда равен 10.
Чтобы исправить можно использовать цикл for вместо while
