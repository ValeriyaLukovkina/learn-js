Конструкция "switch"

1. Напишите if..else, соответствующий следующему switch 


        switch (browser) {
            case 'Edge':
              alert( "You've got the Edge!" );
              break;
          
            case 'Chrome':
            case 'Firefox':
            case 'Safari':
            case 'Opera':
              alert( 'Okay we support these browsers too' );
              break;
          
            default:
              alert( 'We hope that this page looks ok!' );
          }
