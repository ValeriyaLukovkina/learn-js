Циклы while и for

1. Какое последнее значение выведет этот код? Почему?

let i = 3;
while (i) {
alert( i-- );
}

Последнее значение, которое выведется - 1. Потому что когда будет выводиться 1, то значение станет 0 (используется инкрементная форма). while(0) - false, поэтому цикл прервется
