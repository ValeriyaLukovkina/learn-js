Советы по стилю кода

1. Какие недостатки вы видите в стиле написания кода этого примера? 


        function pow(x,n)
        {
            let result=1;
            for(let i=0;i<n;i++) {result*=x;}
            return result;
        }

        let x=prompt("x?",‘‘), n=prompt("n?",‘‘)
        if (n<=0)
        {
            alert(`Степень ${n} не поддерживается, введите целую степень, большую 0`);
        }
        else
        {
        alert(pow(x,n))
        }

Ответ

  1. Во многих местах скобки на новых строчках
  2. Результат условия лучше вынести на другую строку
  3. Отдельные запросы лучше на новых строчках
  4. Много где нет пробелов
