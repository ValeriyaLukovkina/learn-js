Class checking: "instanceof"

1. Почему instanceof в примере ниже возвращает true? Мы же видим, что a не создан с помощью B(). 


        function A() {}
        function B() {}

        A.prototype = B.prototype = {};

        let a = new A();

        alert( a instanceof B );

Ответ: происходит так потому что в примере явно приравняли прототипы между этоми классами. А instanceof сравнивает именно цепочку прототипов и из-за этого выводится true
