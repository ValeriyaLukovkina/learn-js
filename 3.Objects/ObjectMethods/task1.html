Объекты: основы

Методы объекта, "this"

1. Здесь функция makeUser возвращает объект.
Каким будет результат при обращении к свойству объекта ref? Почему?

function makeUser() {
return {
name: "John",
ref: this
};
}
let user = makeUser();
alert( user.ref.name );

Будет выводиться ошибка. Происходит это потому что обращение к user идет через функцию и this становится равным undefined
