1. Что выведет код ниже? 


        let promise = new Promise(function(resolve, reject) {
            resolve(1);
          
            setTimeout(() => resolve(2), 1000);
        });
          
        promise.then(alert);

Ответ: выведеоться только 1, потому что может быть только один результат, остальные будут проигнорированы.
