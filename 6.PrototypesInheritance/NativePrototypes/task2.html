Scheduling

2. Добавьте всем функциям в прототип метод defer(ms), который возвращает обёртку, откладывающую вызов функции на ms миллисекунд.
Например, должно работать так: 


        function f(a, b) {
            alert( a + b );
        }
          
        f.defer(1000)(1, 2); // выведет 3 через 1 секунду.
