Prototypes, inheritance

Prototypal inheritance

4. У нас есть два хомяка: шустрый (speedy) и ленивый (lazy); оба наследуют от общего объекта hamster.
Когда мы кормим одного хомяка, второй тоже наедается. Почему? Как это исправить? 


        let hamster = {
            stomach: [],
          
            eat(food) {
              this.stomach.push(food);
            }
        };
          
        let speedy = {
            __proto__: hamster
        };
          
        let lazy = {
            __proto__: hamster
        };
          
        // Этот хомяк нашёл еду
        speedy.eat("apple");
        alert( speedy.stomach ); // apple
          
        // У этого хомяка тоже есть еда. Почему? Исправьте
        alert( lazy.stomach ); // apple

Ответ: потому что у обоих хомяков один и тоже stomach. Тоесть когда есть один хомяк то происходит заполнение массива stomach в общем объекте hamster, так как у хомяков нет своего stomach. Чтобы это исправить необходимо добавить stomach каждому из хомяков
